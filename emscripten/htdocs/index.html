What's this?

WebN64: Experimental JavaScript/HTML5 version of LazyUSF2 a player for "(Ultra) Nintendo 64 Sound" (based on Mupen64plus). Please be warned that this emulator is not the most stable one, and some music files actually manage to crash it.

Bring your own .miniusf/.usf files by dropping them onto the N64 console (you'll have to drop the .usflib first - before dropping the song files that use it). Respective files can be found for example on modland.com.

2018 by Juergen Wothke (The source code can be found here.)

The music that is currently played as an example is from the game Wipeout64 (Copyright by Psygnosis). May it bring back some good memories of that franchise and I hope there are no objections to using it here.

This page does not use any plugins but is based exclusively on the WebAudio API. The audio samples here are completely calculated within JavaScript by running the Mupen64plus emulator logic.
